# Dreamland

รายละเอียดเกม
- เป็นเกมแนว อาร์เคด เล่นผ่านด่านไปเรื่อยๆ มุมมองเป็นเกมด้านข้าง โดยผู้เล่นจะต้องเอาชนะมอนเตอร์และผ่านด่านไปถึงด้านสุดท้ายให้ได้
- โดยตัวเกมจะมี 3D ผสม 2D พวกตัวละครจะเป็น 3D ส่วนพวกฉากจะเป็น 2D
